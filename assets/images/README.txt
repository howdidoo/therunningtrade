============================================================
THE RUNNING TRADE — Picture Slots
============================================================

Drop your generated JPGs into this folder using the filenames below.
Any slot you have not yet filled will fall back to map.jpg automatically,
so the game is playable from day one even with no other artwork.

Recommended dimensions: 800 × 600 px (4:3) — the picture frame in the
UI is 4:3. Anything close will work; the engine uses object-fit: cover.
For the bitmap aesthetic, generate at 400 × 300 and let the CSS
scale up — it will look correctly chunky.
